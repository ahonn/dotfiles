# dotfiles
My vim/neovim, fish, git, and tmux configuration files

![](http://ahonn-me.oss-cn-beijing.aliyuncs.com/images/zldvf.png)

## Install
```
$ git clone https://github.com/ahonn/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ ./setup.sh
```

## Back up your configuration
```
$ ./script/backup.sh
```
